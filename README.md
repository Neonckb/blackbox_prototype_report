# Blackbox v0 report

The Blackbox v0 prototype testing event was conducted with a cohort of 6 testers who had either profressional expertise or deep experience of software, hardware, and user experience related to fintech. This repository contains:
* Blackbox v0 final report
* Appendix 1: Blackbox prototype testing pack
* Appendix 2: Testing Operations Experience and Reflections
* Appendix 3: Compilation of all quantitative and qualitative feedback received from testers
